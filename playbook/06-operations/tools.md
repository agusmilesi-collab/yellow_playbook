## Stack mínimo viable

Si solo se pudieran usar 5 herramientas, son estas:

1. **Gmail** + WhatsApp para comunicación.
2. **Notion** para base de conocimiento.
3. **Claude Projects** con el playbook cargado.
4. **Figma** para blueprints y entregables visuales.
5. **Google Sheets** para tracking de pipeline.

Todo lo demás es opcional y se puede sumar después.
